How to use mklinkUI

This page shows how to use mklinkUI with Battlefield 3 as an example.

Start mklinkUI.exe and grant it administrative privileges.

Click the top Browse... button and select the folder you wish to move and click OK.

Click the second Browse... button and select the folder you want to move it to (you can create a new) and click OK.

Click Move & Create Link and you're done.
